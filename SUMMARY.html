Run log

01baseline

Unmodified starter code

Byte-level tokenizer (vocab 256), 4-layer attention model, plain Adam, constant LR. Devanagari costs 3 bytes/token — a meaningful chunk of the token budget goes to sub-character units.

bpb 2.3718 — reference
02combined

BPE tokenizer + architecture + optimizer, all at once

256-merge BPE (vocab 512, lossless), wider/deeper model with tied weights and scaled init, AdamW with warmup + cosine + clipping. Clear win, but the three changes are entangled — no way yet to say which one is doing the work.

bpb 2.2833 −3.7%
03ablation

BPE tokenizer alone, everything else reverted to baseline

Isolates the tokenizer against the original 4-layer model and plain Adam. Result lands within 0.0007 of Run 2 — the tokenizer accounts for essentially the entire gain. Architecture and optimizer changes contributed almost nothing at the original learning rate.

bpb 2.2826 −3.8%
04lr sweep

Same v2 config, peak LR 6e-4 → 8e-4

Hypothesis: the larger model from Run 2 needs a higher peak LR to exploit its extra capacity inside a fixed 2,000-step budget. It does — the gap over the tokenizer-only ablation reopens in the model's favor.

bpb 2.2762 −4.0%
05lr sweep

Peak LR 8e-4 → 1e-3

Pushing further still helps, but the gain shrinks (−0.0071 → −0.0042) — a sign of approaching the LR ceiling for this step budget before instability sets in.

bpb 2.272 −4.2%
06final

Batch size 8 → 16, everything else identical to Run 5

The single biggest single-change drop in the log — much larger than batch size alone typically explains. Best-guess mechanism: less gradient noise let the already-tuned lr=1e-3 cosine schedule converge more effectively in the same 2,000 steps. Not a clean ablation: batch 16 at the old LR, and batch 8 at 1e-3, were never tested in isolation, so a batch×LR interaction can't be ruled out. Wall time (543s) was also disproportionately higher than Run 5's (285s), so some of that gap may be machine load rather than pure compute cost — flagged, not swept under the rug.

bpb 2.073 −12.6%
